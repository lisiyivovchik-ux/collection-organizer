# Нормализация БД (1НФ–3НФ)

## 1НФ
Все поля атомарные (в одной ячейке — одно значение), повторяющихся групп нет.

## 2НФ
Связь many-to-many между Items и Tags вынесена в отдельную таблицу ItemTags.

## 3НФ
Справочники вынесены отдельно:
- ItemTypes (типы объектов)
- Tags (теги)
В таблице Items хранятся только данные объекта и внешние ключи (type_id, user_id).
